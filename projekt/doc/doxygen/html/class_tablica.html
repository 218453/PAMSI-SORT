Detailed Description

template<class Type>
class Tablica< Type >

Klasa Tablica, w ktorej odbywa sie zapis dynamiczny elementow.

Implementuje metody interfejsu IRunnable. Zajmuje sie dynamiczna alokacja pamieci. Elastyczna a propos typow wskutek zastosowania szablonow.
