Detailed Description

template<class Type>
class Lista< Type >

Klasa Lista, w ktorej odbywa sie zapis dynamiczny elementow typu int.

Implementuje metody interfejsu IRunnable. Zajmuje sie dynamiczna alokacja pamieci. Lista jest dwukierunkowa oraz posiada mechanizm, dzieki ktoremu mamy dostep zarowno do pierwszego jak i ostatniego elementu.
